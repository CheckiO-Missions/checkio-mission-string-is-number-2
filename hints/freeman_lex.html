So, how my final solution should look like?

You just need to join all previous hints and wrap your code into function expression. Using type hints for input and output data are highly recommended.

Here is the whole solution:

def is_number(val: str) -> bool:

    for char in val:
        if char.isalpha():
            return False
    try:
        float(val)
        return True
    except:
        return False

Here you remove everything, which is not change the checking actually and check the rest with str.isdigit() method. 

def is_number(val: str) -> bool:

    if val.startswith(("+", "-")):
        val = val[1:]

    return val.replace(".", "").isdigit()

Use regular expression module and create a pattern of number. Then compare if the pattern fully match the given string. 

import re


def is_number(val: str) -> bool:

    return bool(re.fullmatch(r'[-+]?(?:\d+\.?\d*|\d*\.\d+)', val))
