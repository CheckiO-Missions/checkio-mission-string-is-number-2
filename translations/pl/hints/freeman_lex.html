Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

💡 Chociaż istnieją liczby, które zawierają literę, w tym konkretnym zadaniu nie są one dozwolone. Proszę więc sprawdzić, czy w liczbie nie ma liter i spróbować przekonwertować ją na typ float. Konwersja ta jest konieczna, ponieważ w ciągu może znajdować się kropka. Jeśli wszystkie te kontrole zakończą się pomyślnie, podany ciąg jest liczbą.

Ok, od czego powinienem zacząć?

Proszę sprawdzić, czy każdy znak w val jest literą. Najpierw za pomocą for pętli iterujemy po podanym ciągu val. 

for char in val:
    ...
Jak wykonać to sprawdzanie?

Proszę sprawdzić, czy znak jest literą za pomocą metody str.isalpha(). Zwraca ona wartość logiczną. Jeśli jej True dla dowolnego znaku, to podany ciąg nie jest liczbą, a Państwa funkcja może już zwrócić False. 

if char.isalpha():
    return False
Co zrobić dalej, jeśli w podanym ciągu nie ma liter?

Tutaj potrzebne jest wyrażenie try/except, aby poprawnie przechwycić ValueError, jeśli val nie może zostać przekonwertowane na float. 

try:
    ...
except ValueError:
    ...
Proszę mi pomóc wypełnić wyrażenie.

Jak już wspomniałem, musi Pan spróbować przekształcić podany ciąg val na float za pomocą wbudowanej funkcji float(). 

try:
    float(val)
    ...

Ponieważ tak naprawdę interesuje Pana proces konwersji, a nie wartość, nie musi Pan przypisywać wyniku do żadnej zmiennej.

Jeśli konwersja się powiedzie, funkcja musi zwrócić True.

Jak wypełnić blok except?

To proste. Jeśli ciąg znaków nie może zostać przekonwertowany na float (zgłaszany jest błąd), funkcja musi zwrócić False.

Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy połączyć wszystkie poprzednie podpowiedzi i zawinąć kod w wyrażenie funkcyjne. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Poniżej znajduje się całe rozwiązanie:

def is_number(val: str) -> bool:

    for char in val:
        if char.isalpha():
            return False
    try:
        float(val)
        return True
    except:
        return False

Jasne, proszę łapać!

Tutaj proszę usunąć wszystko, co nie zmienia sprawdzania i sprawdzić resztę za pomocą metody str.isdigit(). 

def is_number(val: str) -> bool:

    if val.startswith(("+", "-")):
        val = val[1:]

    return val.replace(".", "").isdigit()

Proszę użyć modułu wyrażeń regularnych i utworzyć wzorzec liczb. Następnie proszę porównać, czy wzorzec w pełni pasuje do podanego ciągu znaków. 

import re


def is_number(val: str) -> bool:

    return bool(re.fullmatch(r'[-+]?(?:\d+\.?\d*|\d*\.\d+)', val))
